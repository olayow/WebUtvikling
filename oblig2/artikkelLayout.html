CSS Layout

The important roles of CSS layout in a webpage. The modern design structure we need in todays technologies. All that can be done with CSS coding.. in todays modern era of web development. Structuring a website can sometimes cause annoyance if not done orderly. In the old school era of web development, devs uses a lot of boxes to make a structure of a webpage. Floats can be hard sometimes when creating a CSS layout and using a lot of divs can make your HTML look bad at todays symmantic way of creating elements. The new age of web design has now come up witha solution, Flex and Grid. Using these layout styles will not only make your css much better, it will also have a better responsive design to browsers. Lets take a short decription of different CSS layouts and its usage.

Floats – used a lot before is the older way of layouting a webpage. It still works but it require more things to do and using hacks to have the webpage look the way you want it. You will also end up using a lot of divs in html which is not semantic in todays web development. Flexbox and Grid have similarities when it comes to page layout. The main difference is that flexbox uses one-dimensional layout while grid uses two-dimensional layout (rows and columns). One can use flexbox inside a grid layout. As they say, Grid for layout and Flexbox for content.
